# Vendas API

## Escopo
Projeto de estudos para construção de um sistema de vendas através das aulas ministradas pelo canal no YouTube @gubertdev

## Tecnologias
* Java 21
* Spring Boot
* MongoDB
* Docker
* Swagger/OpenAPI
